# Простой переводчик имён текстовых файлов с транслита на русский для Windows

	Переводит латиницу в кириллицу.

## Как использовать:
1. Скачайте бат файл
2. Переместите его в целевую папку
3. Запустите файл


	Если перевод каких-то букв вас не устраивает просто замените их, открыв файл в текстовом редакторе.
